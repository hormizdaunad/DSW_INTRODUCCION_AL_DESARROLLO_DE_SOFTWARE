Propósito del curso

El propósito fundamental del presente Objeto Virtual de Información (OVI) es permitir que el estudiante pueda comprender de manera fácil e intuitiva el desarrollo del curso de Introducción al Desarrollo de Software, utilizando diferentes archivos multimedia que expliquen la razón de ser del curso.

El Proceso para el desarrollo de software, también denominado ciclo de vida del desarrollo de software es una estructura aplicada al desarrollo de un producto de software. Hay varios modelos a seguir para el establecimiento de un proceso para el desarrollo de software, cada uno de los cuales describe un enfoque diferente para diferentes actividades que tienen lugar durante el proceso. Algunos autores consideran un modelo de ciclo de vida un término más general que un determinado proceso para el desarrollo de software. Por ejemplo, hay varios procesos de desarrollo de software específicos que se ajustan a un modelo de ciclo de vida de espiral.

Durante el desarrollo del curso encontrará lecturas, videos y audios, que hacen referencia a las etapas que debe tener un software, su importancia para aplicarlas y los posibles riesgos que se corre al dejar de aplicar una de ellas.

Objetivos

Objetivo específico 1:

Permitir Identificar fundamentos teóricos para el desarrollo de Software

Objetivo específico 2:

Dar a conocer cada una de las etapas necesarias para desarrollar software, para que el estudiante las identifique y las ponga en práctica

Objetivo específico 3:

Permitir que el estudiante pueda realizar análisis de un Software así como definir especificaciones de requerimientos necesarios para la ejecución de un Software.
